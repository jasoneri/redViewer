## V1.1.1

### ✨ 新增功能

- **自动下滑功能**  
  看预览图右侧设置调速和按钮位置，按钮点击一次自动下滑，再点击就停止下滑。

::: details 使用说明
1. 在阅读页面右侧找到滚动速度设置
2. 调整自动滚动速度
3. 点击自动滚动按钮开始
4. 再次点击停止滚动
:::
